To start local dev:

1. git pull from /Alohomora-App
2. cd /alohomora-backend, execute export client id commands from terminal, and npm start
3. cd /alohomora-react and npm start
4. cd /alohomora-react/src
5. sass --watch App.scss App.css
6. ready for local dev
7. To push changes to github, execute git commmands from /Alohomora-App
8. To push changes to heroku, execute git commands from /alohomora-react and git push heroku master
